# 🏭 WarehouseFlow

## Opis projekta
Informacijski sustav za upravljanje skladišnim poslovanjem.
Razvijen u Blazor .NET 10 s Azure SQL bazom i deployom na Azure Container Apps.

## 🛠️ Tehnologije
- Frontend + Backend: Blazor .NET 10
- Autentikacija: ASP.NET Identity
- Baza podataka: SQL Server / Azure SQL
- ORM: EF Core
- CI/CD: GitHub Actions
- Deploy: Azure Container Apps

## 📦 Entiteti
- Item (Artikl)
- Supplier (Dobavljač)
- Stock (Zaliha)
- Order (Narudžba)
- DeliveryNote (Otpremnica)
- OrderItem (Stavka narudžbe)
- DeliveryNoteItem (Stavka otpremnice)
- User / Role

## 🌿 Grane
- main - stabilna verzija
- develop - razvojna verzija
- feature/* - nove funkcionalnosti
